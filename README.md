# eBird Project: Simpcw Territory
This repository contains the code, small data files, and figures for the eBird Project.

The **full project webpage (Quarto Book)** is hosted here:  
**[eBird Project: Simpcw Territory Webpage](https://kelseyfreitag.github.io/eBird-Modelling-Simpcw/)**

## 1. Project Overview

The project includes:

- Filtering protected areas
- Downloading and handling eBird trend and checklist data
- Mapping eBird trend and checklist data
